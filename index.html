Blackmail

Javascript Gmail client made with JavaScript, HTML, and CSS using Google API

Qué es BlackMail

BlackMail es un cliente de Gmail hecho utilizando únicamente tecnologías web: JavaScript, CSS y HTML. BlackMail utiliza la API de Google para mantener tus correos electrónicos ordenados y categorizados de la misma forma que desde el cliente oficial de Google

Instalación

En primer lugar debes clonar el repositorio: 
$ git clone https://github.com/Brugarolas/BlackMail.git
Después debes de crear un servidor local en el puerto 3000 para acceder a los archivos. Por ejemplo, si estamos utilizando Node.js con Express: 
console.log("Loading server...");
var express = require('express');
var app = express();
var http = require('http').Server(app);

var port = process.env.PORT || 3000;

app.all('/', function(req, res, next) {
    res.header("Access-Control-Allow-Origin", "*");
    res.header("Access-Control-Allow-Headers", "X-Requested-With");
    next();
});

app.use(express.static(__dirname + '/BlackMail'));

http.listen(port, function() {
    console.log('Listening on port ' + port);
});
