What is BlackMail?

BlackMail is a Gmail client created exclusively with web technologies: JavaScript, CSS and HTML. It uses Google API to maintain your emails ordered and classified in the exact same way as in the official Google client.

Installation

First of all you need to clone the following repository: 
$ git clone https://github.com/Brugarolas/BlackMail.git
Next, you need to create a local server on port 3000 to serve files. For example, if we are using Node.js Express: 
console.log("Loading server...");
var express = require('express');
var app = express();
var http = require('http').Server(app);

var port = process.env.PORT || 3000;

app.all('/', function(req, res, next) {
    res.header("Access-Control-Allow-Origin", "*");
    res.header("Access-Control-Allow-Headers", "X-Requested-With");
    next();
});

app.use(express.static(__dirname + '/BlackMail'));

http.listen(port, function() {
    console.log('Listening on port ' + port);
});
